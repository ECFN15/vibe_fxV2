# 📁 `src/data`

## Rôle de ce dossier
Centralise toutes les **données statiques**, les constantes magiques, et les configurations par défaut de l'application.
Cela permet de séparer la donnée (data) de la logique (React).

## Contenu
- `constants.js` : Contient les `FORMATS` (Carré, Story, Pano), les `TEMPLATES` (Film Strip, Polaroid), les listes de polices (`FONT_OPTIONS`), et les banques de `PRESET_CATEGORIES` et `CAMERA_BRANDS`.
